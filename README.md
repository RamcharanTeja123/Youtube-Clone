# YouTube Clone – HTML & CSS


## Description
A simple and responsive YouTube homepage clone created using only HTML and CSS.
This project replicates the UI layout of YouTube, including the navbar, sidebar, and video grid — perfect for practicing front-end design skills.


## Features
* Clean and modern UI inspired by YouTube
* Top navigation bar with logo, search bar, and icons
* Sidebar with menu items
* Video grid with thumbnails, titles, and channel info
* Responsive layout using Flexbox & CSS Grid
* Beginner-friendly folder structure


## Technologies used
  * HTML5
  * CSS3 (Flexbox,Grid)


## Clone Repository
1. Clone the repository: `git clone https://github.com/RamcharanTeja123/youtube-clone-website.git`
2. Open `youtubeclone.html` in a web browser
